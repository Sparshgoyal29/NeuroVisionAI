# 🧠 NeuroVision AI

AI-Powered Brain Tumor Detection System built using React.js, Flask, TensorFlow, and SQLite.

## Features
- MRI Brain Tumor Detection
- PDF Report Generation
- Analytics Dashboard
- Patient History
- Symptom Checker
- AI Assistant

## Tech Stack
- React.js
- Flask
- TensorFlow / Keras
- SQLite
- Chart.js
- jsPDF

## Model
CNN (Convolutional Neural Network)

Classes:
- Glioma
- Meningioma
- Pituitary
- No Tumor
